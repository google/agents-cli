# ADK Skills

Development skills for building agents with [Google ADK](https://google.github.io/adk-docs). Install into any coding agent via [`npx skills`](https://github.com/vercel-labs/skills).


## Skills

| Skill | Description |
|-------|-------------|
| `agents-cli-workflow` | Development lifecycle, code preservation rules, model selection |
| `agents-cli-adk-code` | Python API reference — agents, tools, orchestration, callbacks, state |
| `agents-cli-scaffold` | Project scaffolding via Agents CLI |
| `agents-cli-eval` | Evaluation methodology — metrics, evalsets, LLM-as-judge, trajectory scoring |
| `agents-cli-deploy` | Deployment — Agent Engine, Cloud Run, CI/CD, secrets |
| `agents-cli-publish` | Gemini Enterprise registration |
| `agents-cli-observability` | Observability — Cloud Trace, Cloud Logging, third-party integrations |
